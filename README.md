# PostgreSQL Development Assistant Plugin

[![SASMP v1.3.0](https://img.shields.io/badge/SASMP-v1.3.0-blue.svg)](https://github.com/pluginagentmarketplace)
[![PostgreSQL](https://img.shields.io/badge/PostgreSQL-16+-336791?logo=postgresql)](https://www.postgresql.org)

Comprehensive PostgreSQL database plugin with 8 specialized agents and 12 Golden Format skills for mastering PostgreSQL.

## Agents (8)

| Agent | Description |
|-------|-------------|
| pg-fundamentals | Master PostgreSQL basics |
| pg-sql-mastery | SQL excellence |
| pg-data-types | Type system mastery |
| pg-indexing | Index strategies |
| pg-performance | Performance tuning |
| pg-administration | DBA operations |
| pg-replication | High availability |
| pg-advanced | PL/pgSQL and extensions |

## Skills (12)

- Core: pg-basics, pg-ddl, pg-dml, pg-queries
- Performance: pg-indexes, pg-performance, pg-transactions
- Admin: pg-backup, pg-replication, pg-security
- Advanced: pg-functions, pg-extensions

## Commands (4)

- /pg-check - Database health checks
- /pg-query - Query with EXPLAIN ANALYZE
- /pg-optimize - Performance optimization
- /pg-backup - Create backups
