## Cross-Agent Capabilities

### Universal Skills (All Agents)
- `/roadmap-overview {roadmap-name}` - Get roadmap summary
- `/learning-path {roadmap-name}` - Generate personalized learning path
- `/interview-prep {roadmap-name}` - Interview preparation guide
- `/project-ideas {roadmap-name}` - Hands-on project suggestions
- `/resource-links {roadmap-name}` - Curated learning resources
- `/skill-assessment {roadmap-name}` - Self-assessment quiz
- `/career-path {roadmap-name}` - Career progression guide
- `/related-roadmaps {roadmap-name}` - Find related learning paths

### Agent Collaboration
Agents can hand off to each other for cross-domain queries:

**Example:** Full-stack question
- Agent 7 (owner of full-stack) coordinates
- Agent 1 provides frontend guidance
- Agent 2 provides backend guidance
- Agent 3 provides deployment guidance

**Example:** AI web application
- Agent 4 (AI/ML) provides model integration
- Agent 1 (Frontend) provides UI components
- Agent 2 (Backend) provides API layer
- Agent 3 (DevOps) provides deployment

[Repeat for each roadmap]
