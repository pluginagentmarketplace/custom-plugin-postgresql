# Learn - Personalized Developer Roadmap

Start your personalized learning journey with the ultrathink plugin. Choose your role, experience level, and learning objectives.

## Command Usage

```
/learn
```

## Interactive Learning Path Selection

When you use this command, you'll be guided through:

### 1. **Role Selection**
Choose from 7 main specializations:
- **Frontend & Web Development** - HTML, CSS, JavaScript, React, Vue, Angular
- **Backend & API Development** - Node.js, Python, Java, Go, PHP
- **DevOps & Infrastructure** - Docker, Kubernetes, Terraform, AWS
- **Data, AI & Machine Learning** - ML, Deep Learning, MLOps
- **Mobile & Gaming** - iOS, Android, React Native, Flutter
- **Database & Architecture** - SQL, NoSQL, System Design
- **Specialized & Management** - Full-stack, QA, Engineering Manager, Product Management

### 2. **Experience Level**
- **Beginner** - Starting from scratch
- **Intermediate** - Some experience, looking to level up
- **Advanced** - Deep expertise, specialized focus

### 3. **Learning Goal**
- **Career Change** - Complete career transition
- **Skill Enhancement** - Add specific skills
- **Interview Prep** - Preparing for job interviews
- **Project-Based** - Learning through projects
- **Certification** - Pursuing industry certifications

## Learning Path Structure

Each learning path includes:
- **Foundation Phase**: Core concepts and fundamentals
- **Intermediate Phase**: Advanced techniques and patterns
- **Advanced Phase**: Expert-level topics and specialization
- **Practical Projects**: Real-world applications
- **Assessment**: Knowledge checks and quizzes

## Timeline Estimates

- **Beginner Path**: 3-6 months (12-20 hours/week)
- **Intermediate Path**: 2-4 months (10-15 hours/week)
- **Advanced Path**: 2-3 months (8-12 hours/week)
- **Specialized Path**: 1-2 months (5-10 hours/week)

## Learning Resources

Each path includes:
- Video tutorials
- Written guides
- Code examples
- Practice exercises
- Real-world projects
- Community support

## Next Steps

After selecting your learning path:
1. Review the curriculum overview
2. Start with the first module
3. Complete lessons and projects
4. Track your progress
5. Use `/assess` for self-evaluation
6. Build projects with `/projects`
7. Explore related skills with `/browse-agent`

## Tips for Success

- Start with foundational concepts
- Build projects as you learn
- Review and reinforce regularly
- Practice coding daily
- Join community discussions
- Don't skip the projects
- Focus on understanding concepts
- Build a learning portfolio

- Your preferred technologies
