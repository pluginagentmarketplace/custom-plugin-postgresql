# PostgreSQL Optimize Command

Analyze and optimize database performance.

## Usage
```
/pg-optimize [table-name]
```

## Actions
1. ANALYZE tables
2. Identify missing indexes
3. Detect slow queries
4. Suggest configuration changes
5. Find unused indexes

## Recommendations Generated
- Index creation SQL
- Configuration parameters
- Query rewrites
